# Metro-Traffic-Analytics-Excel-Dashboard
Interactive Excel dashboard analyzing traffic patterns and weather impact.


## Project Overview

This project analyzes traffic patterns on the Metro Interstate using historical traffic and weather data. The goal was to identify peak congestion hours, weather impact on traffic flow, and differences between weekday and weekend traffic.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Slicers
* Data Cleaning & Transformation

## Dashboard Features

* KPI Cards showing key traffic metrics
* Traffic trend by hour of day
* Weather impact analysis
* Weekly and monthly traffic patterns
* Interactive slicers for filtering data

## Key Insights

* Traffic peaks during morning and evening commute hours
* Weather conditions such as rain slightly reduce traffic volume
* Weekday traffic is significantly higher than weekend traffic
